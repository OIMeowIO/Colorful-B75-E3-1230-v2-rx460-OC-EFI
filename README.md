# 七彩虹 战斧B75 E3-1230 v2 - rx460 4g
我的七彩虹战斧B75主板安装黑苹果的EFI(OC)文件

## 已知反馈
- rx460 DVI输出可能紫屏(花屏)
- 没有进行USB定制 (可能等会就有了)
- 可以使用快速启动
- 可以使用休眠
- 可开启VT-x (不支持VT-d?)

## 硬件配置
- CPU	Intel E3 1230 v2
- 主板	七彩虹C.B75K
- 显卡	蓝宝石RX460 4g
- 内存	双通道 8g
- 网卡	USB RTL8188CU

## 关于我的网卡 
- 不支持蓝牙
- 需要安装对应的网卡驱动
- 不能用隔空投送
- 驱动安装方面可以参考另一个仓库 
 github.com/chris1111/Wireless-USB-OC-Big-Sur-Adapter

## EFI文件说明
将根目录下的EFI文件夹放到对应的EFI分区即可
- 使用了Monterey主题
- 注入了基础驱动
- 注入了我的USB网卡驱动（需要配合软件 否则无效果）
- 删除了-v参数（可自行添加）
